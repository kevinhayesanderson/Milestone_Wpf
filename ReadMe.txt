1.  Run Setup.msi as Administrator, this installs the Milestone app and creates a shortcut to the app on the desktop.

2.  On Running Milestone shortcut and on clicking the "Open Files" button, the user can select multiple '.xlsb' files to generate corresponding charts.

3.  App opens generated HTML file directly on user's default browser.

4.  App creates 'Milestone' folder in the same directory as '.xlsb' file.

5.  'Milestone' folder contains generated HTML file and 'Script' folder(support files for HTML).

6.  In the case of multiple '.xlsb' files, the User is recommended to have the files in the same folder, so that there is a single 'Milestone' output folder.
